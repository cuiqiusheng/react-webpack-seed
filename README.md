# webpack4 seed project

> Don't panic!

### 目录结构说明
```
.
├── README.md               说明文档
├── dist                    打包文件
├── maps                    地图文件
├── node_modules            依赖包
├── package.json            项目配置信息
├── postcss.config.js       postcss配置
├── src                     源代码
├── webpack-config          webpack插件和加载器
├── webpack.config.js       webpack配置文件
└── yarn.lock               yarn依赖管理
```

### 启动开发环境
1. 安装依赖：```npm install``` 或 ```yarn install```
2. 启动开发环境：```npm start```
3. 浏览器访问：[点这里](http://127.0.0.1:9951)

### 打包
```npm run build```
